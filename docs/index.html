Features

Built for privacy. Designed for you.

DELIAH combines the power of modern language models with a completely local architecture.

🔒

100% Local

Every conversation stays on your machine. No cloud, no tracking, no data leaks. Ever.

🧠

Smart Routing

DELIAH detects what you need -- coding, planning, or general chat -- and routes to the best model.

🎨

Personality

A unique AI character with moods, traits, and a consistent voice across all your conversations.

📋

Memory

Remembers your preferences, past conversations, and important details -- all in a local SQLite database.

🎤

Voice Ready

Built-in support for speech-to-text and text-to-speech with local and cloud options.

🔌

Extensible

Plugin system for Minecraft, Discord, Arduino, smart home, OBS, and more.

Setup Guide

Up and running in minutes

Three steps to your own local AI assistant.

1

Install Ollama

Download Ollama and pull the models: ollama pull qwen2.5:7b and ollama pull deepseek-coder:6.7b.

2

Start the Backend

Clone the repo, install dependencies, and run: python -m uvicorn backend.api.main:app --reload.

3

Launch the Desktop App

Run npm run dev in the windows app directory. The DELIAH window opens and connects automatically.

Architecture

How it works

A clean, modular architecture that keeps everything local.

Desktop App (React + Tauri) | | HTTP / WebSocket v DELIAH Backend (Python + FastAPI) | | HTTP v Ollama (Local LLM Runtime) | v AI Models (Qwen, DeepSeek, Llama) Android Companion (Flutter) | | HTTP / WebSocket v DELIAH Backend (same network)
